# Resetting Styles
Обнуление CSS для сайтов – инструмент для стандартизации стилей и их очистки для нового дизайна сайтов.
Обнуление css
## Создайте отдельный файл `Reset.css` и вставьте все из файла Reset.css.
При необходимости, можно редактировать цвета и шрифты

## Поддерживаемые браузеры
Данный набор стилей для сброса CSS совместим со следующими браузерами:

- **Google Chrome** (последние версии)
- **Mozilla Firefox** (последние версии)
- **Safari** (последние версии)
- **Microsoft Edge** (последние версии)
- **Opera** (последние версии)

Рекомендуется использовать последние версии браузеров для обеспечения максимальной совместимости и поддержки современных стандартов CSS.
